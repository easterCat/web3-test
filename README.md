# 目录结构

* controllers/ - 定义你应用的路由和其他的逻辑
* helpers/ - 可以被项目各部分所调用的功能函数和代码
* middlewares/ - Express 中间件，将要处理在进入路由之前的请求
* models/ - 表示数据，实现业务逻辑和处理存储
* public/ - 包含所有的静态文件，像图片、样式和脚本
* views/ - 提供模板文件，模板文件将会在你路由中进行渲染和使用
* tests/ - 测试在其他文件夹的的代码
* app.js - 初始化应用
* package.json - 你应用所依赖的特定版本的包
